Отчет по лабораторной работе № 7

«Связь методического аппарата с уровнем ПВК. Оценка физиологической цены»

Студент

ФИО: Алексеев Леонид Алексеевич группа: P3323

1. Цель и задачи

Определить критерии ПВК для выбранной профессии программиста как комбинации показателей из тестов (ЛР3–ЛР6) с весами, направлениями и «срезами». Провести оценку физиологической цены деятельности по данным нейроинтерфейсов (импорт усреднённых показателей ДО/ВО ВРЕМЯ/ПОСЛЕ) и сравнить изменения.

2. Методика

HTML/CSS/JavaScript. Критерий формируется из показателей тестов, каждому назначается вес, направление (↑/↓) и «срез». Нормирование — по полу и возрасту ±5 лет текущей выборки через перцентили. Итог критерия — взвешенное среднее перцентилей (с учётом направления), обнуление при нарушении «среза». Итог по профессии — взвешенная сумма критериев. Физиология — импорт средних значений ДО/ВО ВРЕМЯ/ПОСЛЕ, расчёт относительных изменений и факт возврата к фону (±5%).

3. Результаты

  • Конструктор критериев ПВК и расчёт итогов по профессии.
  • Интеграция показателей из ЛР3–ЛР6 и нормирование по выборке.
  • Импорт физиологических показателей и анализ динамики ДО/ВО ВРЕМЯ/ПОСЛЕ.

4. Выводы

Определены критерии ПВК, рассчитаны интегральные индексы и выявлены изменения физиологических показателей в ходе тестирования. Предложение по интеграции нейроинтерфейсов: использовать WebUSB/WebBluetooth/локальный агент для записи сигналов и передачу агрегированных метрик через API в модуль ЛР7.
